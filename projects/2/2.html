    Optical Image Recognition and Analysis

    This project was created under the YWCC Capstone Program during the Spring semester of 2022. I worked with one of the sponsors, Case Medical, alongside four others to produce software that would help aid in the quality assurance process during manufacturing.

    The objective that we were given by our sponsor was to create a software platform that would utilize machine learning and image recognition for quality assurance. This would help eliminate possible errors in medical cases during and after the manufacturing process. The tools used were medical grade trays, provided by Case Medical and a camera for the photos. The software was written in Python, HTML, CSS, and JavaScript. We referenced different machine learning libraries: OpenCV, TensorFlow, and Keras. Additionally, we used different data analytic libraries: Numpy, Matplotlib, and Pandas.

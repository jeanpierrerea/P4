    TFT API Analyzer

    TFT API Analyzer was the capstone project for the Fall 2021 Semester. I collaborated with a team of 5 to develop a video game analysis web app hosted on a LAN network that would provide the end user with relevant statistics. Our project was built for the popular video game, Teamfight Tactics, created by Riot Games.

    I created queues and exchanges in Python with the RabbitMQ framework to facilitate messaging between systems. I created a Python program that would retrieve JSON data using HTTP requests through the Riot Games API. I also designed a landing page for data visualization with HTML and CSS.
